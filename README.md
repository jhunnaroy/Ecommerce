# 🛒 MERN E-Commerce Application

A full-featured E-Commerce web application built using the **MERN Stack (MongoDB, Express, React, Node.js)** with authentication, payment integration, admin dashboard, and product management.

---

## 🌐 Live Demo

👉 **Frontend URL:**
https://frontend-vo3n.onrender.com

---

## 🚀 Tech Stack

### Frontend:

* React.js
* Redux
* Axios

### Backend:

* Node.js
* Express.js

### Database:

* MongoDB

### Authentication:

* JWT & Cookies

### Payment Gateway:

* Stripe

### Image Upload:

* Cloudinary

---

## ⚙️ Prerequisites

Make sure you have installed:

* Node.js
* MongoDB
* Git

Basic knowledge of:

* React.js
* Node.js
* Express.js
* MongoDB
